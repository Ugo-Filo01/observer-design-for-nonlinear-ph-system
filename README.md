# Observer Design for Nonlinear Port-Hamiltonian Systems

This repository contains the implementation and simulation code for observer design applied to nonlinear port-Hamiltonian systems, developed during the research internship at FEMTO-ST laboratory, Besançon, France.

## Structure

- **Mechanical Case**: Implementation of observer design for mechanical port-Hamiltonian systems
- **Electromechanical Case**: Implementation for electromechanical systems (in progress)
- **Documents**: Documentation and additional resources
